)what is prototype chain :-
It is a core concept of javascript inheritance model. 
It allows object to inherit properties and methods from other object 
the prototype chain enables inheritance in javascript. 
If a property isnt found on an object , javascript looks up its prototype chain. 
The prototype of an object instance can be accessed with Object.getPrototypeOf(obj) or _proto_.

)what is difference between null and undefined?
| Feature         | `undefined`                  | `null`                    |
| --------------- | ---------------------------- | ------------------------- |
| Meaning         | Not assigned / missing value | Intentionally empty value |
| Set by          | JavaScript (default)         | Developer                 |
| Type            | `undefined`                  | `object` (JS bug ⚠️)      |
| Equality (`==`) | `null == undefined` → ✅ true | Same                     |
| Strict (`===`)  | ❌ false                      | ❌ false                |
