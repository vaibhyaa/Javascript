)what is difference between window and document?
| Feature    | `window` 🌐                    | `document` 📄                         |
| ---------- | ------------------------------ | ------------------------------------- |
| Represents | Browser window                 | Web page (HTML DOM)                   |
| Type       | Global object                  | DOM object                            |
| Access     | Top-level object               | Property of `window`                  |
| Usage      | Browser APIs, global variables | Access & manipulate HTML              |
| Example    | `alert()`, `setTimeout()`      | `getElementById()`, `querySelector()` |





)what is event ?
An event is an action that happens in the browser.
An event is something that happens in the browser, like a click, key press, or page load.
An event handler is a function that runs when an event occurs.
button.addEventListener("click", () => {
  console.log("Button clicked");
});
Here:
click → event
function → event handler
